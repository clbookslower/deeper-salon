PART 1 — 도구 설치 (윈도우, 처음부터)

1-1. Node.js 설치 (클로드 코드의 토대)

클로드 코드는 Node.js 위에서 돕니다. 먼저 이것부터.

  1. 웹브라우저에서 https://nodejs.org 접속
  2. 큰 버튼 두 개 중 "LTS"(권장, 왼쪽) 버튼 클릭 → 설치 파일(.msi) 다운로드
  3. 받은 파일 더블클릭 → Next만 계속 눌러 설치 (기본값 그대로, 체크 건드리지 않음)
  4. 설치 끝나면 잘 깔렸는지 확인:
    • 시작 메뉴에서 PowerShell 검색 → 열기
    • 까만 창에 아래를 입력하고 Enter:
    node --version
    v20.x.x 같은 숫자가 나오면 성공. (숫자가 나오면 버전은 무엇이든 괜찮습니다)
❗ 숫자가 안 나오고 빨간 글씨가 나오면: PowerShell 창을 완전히 닫고 다시 열어서 한 번 더. 그래도 안 되면 PART 4-①.

1-2. 클로드 코드 설치 + 로그인

PowerShell 창에서(방금 그 까만 창) 아래를 입력:

npm install -g @anthropic-ai/claude-code
  • 글자가 주르륵 흐르다 멈추면 설치 완료.
  • 확인: claude --version 입력 → 숫자가 나오면 OK.

이제 클로드 코드를 처음 실행합니다.

  1. PowerShell에 입력: claude
  2. 처음이면 로그인 안내가 뜹니다. 화면 안내대로 진행:
    • 브라우저가 열리며 로그인 화면 → 야간비행님 계정(또는 DEEPER 계정)으로 로그인
    • "권한 허용" 누르면 창에 "로그인됨" 표시
  3. 로그인되면 클로드 코드 대화창이 뜹니다. 일단 exit 입력하거나 Ctrl+C 두 번으로 빠져나옵니다. (다음 단계에서 다시 켭니다)
💡 어느 계정으로 로그인할지: DEEPER 업무 전용으로 쓸 계정 하나를 정해두면 깔끔합니다. (경선님과 상의)

1-3. deeper-os 플러그인 설치  ⭐ 가장 중요 — 순서·형식 그대로

여기가 사람들이 가장 많이 막히는 곳입니다. 주소 형식을 정확히 지키면 한 번에 됩니다.

PowerShell에서 claude 입력해 클로드 코드를 켠 뒤, 대화창에 아래 세 줄을 한 줄씩 차례로 입력합니다. (한 줄 넣고 Enter, 끝나면 다음 줄)

/plugin marketplace add https://github.com/clbookslower/deeper-os.git
/plugin install deeper-os@deeper-os
/reload-plugins

꼭 지킬 점 (맥가이버 메모)

  • 첫 줄은 반드시 https://...git 전체 주소로. clbookslower/deeper-os 같은 짧은 형식으로 넣으면 클로드가 다른 방식으로 내려받으려다 실패합니다. (참여자 대부분이 여기서 걸림)
  • 두 번째 줄(install)이 조용히 아무 말 없이("(no content)") 끝나 보이면, 메뉴로 설치하세요: /plugin 입력 → 메뉴에서 Marketplaces → deeper-os → Install for you 선택
  • 설치되면 /deeper-os:setup 같은 명령이 쓸 수 있게 됩니다.
✅ 확인: 대화창에 / 만 쳤을 때 목록에 deeper-os:setup, deeper-os:meaning 등이 보이면 성공.

1-4. (선택) 옵시디언 설치 — 눈으로 보기 좋게

볼트는 사실 그냥 폴더라서 옵시디언 없이도 클로드 코드만으로 다 됩니다. 다만 노트를 예쁘게 보고 연결망(그래프)으로 한눈에 보고 싶다면 옵시디언이 잘 맞습니다.

  1. https://obsidian.md → Download → 윈도우용 설치
  2. 설치 후 볼트는 PART 2에서 만든 폴더를 "기존 폴더 열기(Open folder as vault)"로 가리키면 됩니다.
옵시디언은 보기 도구일 뿐, 세팅·작업은 클로드 코드가 합니다. 부담 없이 나중에 깔아도 됩니다.

PART 2 — 두 개의 볼트 만들기

2-A. 개인 세컨브레인 (야간비행님만의 것)

본인의 글·목소리·브랜드를 쌓는 사유 자산입니다. 야간비행님은 DEEPER 파운더이자 저자(『밤에 날아야 보이는 것들』)라, 이 개인 볼트가 본인 IP·브랜드의 뿌리가 됩니다.

  1. 폴더 하나 만들기 — 예: 문서 안에 야간비행_세컨브레인 폴더 생성.
    ⚠️ 이 폴더는 OneDrive 동기화 폴더 바깥에 두는 걸 권합니다(개인 자산이라 공유 불필요). C:\야간비행_세컨브레인 처럼.
  2. PowerShell에서 그 폴더로 이동 후 클로드 코드 실행: 
    cd C:\야간비행_세컨브레인
claude
  3. 클로드 코드 대화창에서: 
    /deeper-os:setup
  4. 대화로 CLAUDE.md · 폴더 구조(문법/논리/수사) · 의미의 축을 만들어 줍니다. 질문에 편하게 답하면 됩니다.
    • 이미 만든 노트가 있으면 건드리지 않고 없는 것만 추가합니다(안전장치 내장).

첫 사용 추천 흐름 (천천히, 며칠에 걸쳐도 됨)

  • 먼저 "의미" — meaning: 왜 이 일을 하는지 의미의 축을 길어 올림 ("내 일의 의미를 찾고 싶어")
  • 그다음 목소리 — voice: 본인 글을 읽혀 언어 지문 분석 ("내 문체 분석해줘")
  • 지형도 — topography: 뭘 써야 할지 추천 ("뭐부터 쓸까?")
  • 브랜딩 — branding: 이상적 독자·IP 확장·오퍼 설계
스킬은 말로 부르면 알아서 옵니다. 명령어를 외울 필요 없어요.

2-B. DEEPER 공동운영  ⭐ 경선님과 함께 보는 폴더

이게 두 분이 출판·커뮤니티를 함께 추진하는 핵심 채널입니다. 경선님이 미리 만들어 둔 DEEPER_공동운영 폴더를 OneDrive로 공유하고, 야간비행님 컴퓨터에서 받습니다.

경선님이 먼저 (이미 준비됨)

  1. ✅ 폴더는 이미 OneDrive에 있습니다: C:\Users\clboo\OneDrive\DEEPER_공동운영
  2. 파일 탐색기에서 그 폴더 우클릭 → 공유 → 야간비행님 이메일로 편집 가능으로 공유. (이 한 단계만 남음)

야간비행님 컴퓨터에서

  1. 받은 공유 알림(메일/OneDrive)에서 "내 OneDrive에 추가" 또는 "바로 가기 추가"를 누릅니다.
  2. 잠시 뒤 야간비행님 OneDrive 안에 DEEPER_공동운영 폴더가 동기화됩니다.
  3. PowerShell에서 그 폴더로 가서 클로드 코드 실행: 
    cd "$env:USERPROFILE\OneDrive\DEEPER_공동운영"
claude
    (폴더 위치가 다르면 실제 경로로. 탐색기에서 폴더 주소창을 복사해 cd "붙여넣기" 하면 정확합니다)
  4. 클로드에게 인사처럼 물어보세요: 
    여기가 어떤 작업공간이야?
    클로드가 .claude/CLAUDE.md를 읽고 우리 팀·규칙·맥락을 이해한 상태로 답하면 성공입니다.
📂 이 폴더 안에는 이미 다 차려져 있습니다:
_시작하기.md(여기부터) · _대시보드.md(현황 한 장) · _협업규칙.md(약속) · 폴더 5개(허브사이트·8권진행·살롱·디자인·레터) · 에이전트 팀 5명.

PART 3 — 함께 일하는 법 (설치 후 계속)

도구를 깔았으니, 이제 출판과 커뮤니티를 어떻게 같이 굴릴지입니다.
핵심 한 문장: "같은 폴더를 보고, 같은 팀을 부르고, 한 파일은 한 사람이 만진다."

3-1. 동기화는 어떻게 되나 (원리)

  • DEEPER_공동운영 폴더는 OneDrive가 두 컴퓨터를 자동으로 맞춰 줍니다.
  • 경선님이 저장 → 잠시 뒤 야간비행님 화면에도 똑같이. 반대도 같음.
  • 그래서 이메일로 파일 주고받기, 카톡으로 최신본 찾기가 사라집니다. 항상 같은 한 벌.

3-2. 단 하나의 약속 — "한 파일은 한 사람이"

동기화의 유일한 주의점은, 같은 파일을 동시에 두 사람이 고칠 때입니다(충돌 사본이 생김).

  • 어떤 파일을 한동안 만질 거면 → _협업규칙.md의 '지금 만지는 중' 표에 한 줄 적기. 끝나면 지우기.
  • 빠른 한마디는 카톡, 남길 결정·요청은 글로(04_레터_소통/ 또는 _대시보드.md).

3-3. 같은 에이전트 팀을 둘이 함께

공유 폴더에서 클로드 코드를 열면 두 분 다 같은 팀을 부릅니다:

① node/claude 명령이 빨간 글씨로 안 됨 PowerShell을 완전히 닫고 다시 열기. 그래도면 컴퓨터 재시작 후 1-1부터. (설치 후 창이 옛 상태라 그런 경우가 대부분)
② 플러그인 설치가 조용히 안 됨 / 목록에 안 보임 /plugin 메뉴 → Marketplaces → deeper-os → Install for you (user scope). 그래도면 첫 줄 주소가 https://...git 전체였는지 확인.
③ "host key" / SSH 오류 짧은 주소로 넣은 경우입니다. 마켓플레이스를 지우고 https://github.com/clbookslower/deeper-os.git 전체 주소로 다시.
④ OneDrive 초록 체크가 안 됨 / 안 보임 작업표시줄 OneDrive 아이콘 클릭 → 동기화 상태 확인. 로그인 계정이 공유받은 계정인지 확인. 파란 화살표면 올리는 중이니 잠깐 기다림.
⑤ ... (충돌본) 파일이 생김 같은 파일을 둘이 동시에 고쳐서 생긴 것. 당황 금지. 둘이 한 번 보고 최신 내용을 한쪽으로 합친 뒤 충돌본 삭제. (앞으로 '지금 만지는 중' 표 쓰면 거의 안 생김)
⑥ 그 외 무엇이든 맥가이버를 부르세요. 공유 폴더에서 클로드 코드를 켜고: 맥가이버, OneDrive가 초록불이 안 떠요 처럼. 원인과 해결을 같이 알려줍니다.
