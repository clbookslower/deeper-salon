  • 매일 질문 탐구

윈도우 노트북에서 시작합니다. 순서대로 하면 오늘 안에 완료됩니다.

멤버 가이드와 겹치는 부분은 빠르게: Obsidian 설치, OneDrive 폴더 생성, 폴더 구조 만들기 — 방법은 멤버 가이드와 동일합니다. 여기서는 경선 씨에게만 다른 부분을 상세히 설명합니다.

STEP 1 — Obsidian 설치 + OneDrive 연결

멤버 가이드 STEP 0~2와 동일합니다. 다만 폴더 이름을 아래처럼 잡으세요.

OneDrive/ └── Deeper Salon Leader/ ← 이 이름으로 만들기 (멤버와 구분) └── deeper-leader/ ← Vault 이름
왜 멤버 Vault와 분리하는가: 나중에 멤버들이 Vault를 공유하거나 비교할 일이 생길 수 있습니다. 리더의 멤버 관찰 노트와 개인 탐구 노트가 뒤섞이지 않도록 처음부터 분리합니다.

STEP 2 — 리더 전용 폴더 구조 만들기

멤버 구조 위에 SALON 폴더를 추가합니다.

deeper-leader/ │ ├── 00-Daily/ ← 나의 일간 노트 (멤버와 동일) ├── 01-Grammar/ ← 나의 5월 해체 데이터 │ ├── Questions/ │ ├── Raw-Text/ │ └── Insights/ ├── 02-Logic/ ← 나의 6월 구조 설계 ├── 03-Rhetoric/ ← 나의 7월 발화 ├── Templates/ ← 템플릿 보관 │ ├── SALON/ ← 리더 전용 운영 폴더 │ ├── Sessions/ ← 세션별 준비·진행·후기 │ ├── Members/ ← 멤버 7명 관찰 노트 │ ├── Group-Insights/ ← 그룹 전체에서 보이는 패턴 │ └── Curriculum/ ← 커리큘럼 업데이트·메모 │ └── CONTENT/ ← 살롱에서 나온 것 → 내 콘텐츠로 └── Ideas/

STEP 3 — 플러그인 설치

멤버 가이드와 동일하게 Templater + Daily Notes + Dataview 3개 설치. 설치 방법은 멤버 가이드 STEP 4 참조.

추가로 하나 더:

+ Calendar 플러그인
리더에게 권장 추가 플러그인
  • Community plugins → "Calendar" 검색 → 설치 → 활성화 왼쪽 사이드바에 달력이 생깁니다. 날짜를 클릭하면 그날 Daily Note로 바로 이동합니다. 세션 날짜 전후를 빠르게 오갈 때 유용합니다.

STEP 4 — Daily Note 템플릿 만들기

Templates 폴더에 Daily-Template.md 파일 생성 후 아래 내용을 붙여넣기. 멤버 템플릿에 리더 섹션이 추가됩니다.

--- date: <% tp.date.now("YYYY-MM-DD") %> week: <% tp.date.now("W") %> type: daily role: leader --- # <% tp.date.now("MM월 DD일 (ddd)") %> ## 오늘의 질문 > ## 모닝페이지 메모 ## 오늘의 키씬 - [ ] ## Claude와 탐구한 것 ## 오늘 발견한 것 ## 감사 + 비전 - 감사: - 비전: --- ## 리더 시선 (선택) (멤버들에게서 보이는 것, 세션 준비 메모, 살롱에서 떠오른 것)

Daily Notes 설정 → Template file location에 Templates/Daily-Template 입력. Templater 설정 → Template folder location에 Templates 입력.

STEP 6 — 오늘 밤: 나의 첫 질문

경선 씨가 이 질문을 먼저, 그리고 깊이 파고들수록 5/7 세션이 달라집니다.

매주 주말 — 멤버 관찰 리뷰 (20분)

7명의 파일을 순서대로 열어 아래 질문을 스스로에게 던집니다. Claude에게 물어봐도 됩니다.

  • 이번 주 세션에서 이 멤버가 말한 것 중 가장 기억에 남는 것은?
  • 이 멤버가 아직 꺼내지 못하고 있는 것이 보이는가?
  • 이 멤버의 강점이 서서히 드러나고 있다면, 그것은 무엇인가?
Claude에게 멤버 파악 요청하는 법: "살롱 운영" 프로젝트에서 특정 멤버의 세션 발언들을 붙여넣고 "이 사람이 아직 말하지 못하고 있는 것, 그리고 강점으로 보이는 것을 짚어줘" 라고 요청합니다.

지금 OneDrive에 쌓은 것은 맥미니로 그대로 이어집니다. 전환 자체는 1시간이면 됩니다.

맥미니 도착 당일 할 것

  • OneDrive 데스크탑 앱 설치 + 로그인

    지금 윈도우에서 쌓은 모든 Obsidian 파일이 자동으로 동기화됩니다.

  • obsidian.md에서 Mac 버전 다운로드 + 설치

    Obsidian 열기 → "Open folder as vault" → OneDrive/Deeper Salon Leader/deeper-leader 폴더 선택. 모든 노트와 설정이 그대로 열립니다.

    ✅ 플러그인 설정(.obsidian 폴더)도 OneDrive에 저장돼 있어서 재설정 불필요.
  • Claude 앱 Mac 버전 설치

    claude.ai → 다운로드 또는 App Store. Pro 계정으로 로그인하면 Projects가 그대로 이어집니다.

맥미니 전환 시 추가 고려사항:
Claude Code를 쓰게 되면 Mac 환경이 훨씬 수월합니다. 터미널이 기본 탑재돼 있고, Homebrew로 개발 도구 설치가 간단합니다. Level 4 확장을 계획하고 있다면 맥미니가 맞는 선택입니다.
