지금 당장 할 것

윈도우 노트북에서 시작합니다. 아래 순서대로 하면 오늘 안에 모든 셋업이 완료됩니다. 반드시 PC에서 진행하세요 — 아이패드·아이폰은 Obsidian Vault를 OneDrive에 연결할 수 없습니다.

⚠️ PC(Windows/Mac) 전용: 아이패드·아이폰 Obsidian은 iCloud 저장만 지원합니다. OneDrive 연결은 데스크탑 앱에서만 가능합니다. 셋업 완료 후 아이패드는 열람·가벼운 메모용으로 씁니다.

STEP 0 — Obsidian 설치

  • 브라우저에서 obsidian.md 접속

    가운데 "Download" 버튼 클릭. Windows라면 .exe, Mac이라면 .dmg 파일을 선택합니다.

  • 설치 파일 실행

    Windows: 다운로드된 .exe 파일을 더블클릭 → "다음" 클릭하며 설치 완료.
    Mac: .dmg 파일을 열어 Obsidian 아이콘을 Applications 폴더로 드래그.

    ✅ 설치 후 Obsidian을 실행하면 "Open vault" 화면이 나타납니다. 이 화면이 보이면 성공입니다.

STEP 1 — OneDrive 확인 및 폴더 생성

  • OneDrive 데스크탑 앱이 실행 중인지 확인

    Windows: 화면 오른쪽 아래 작업표시줄에 흰색 구름 ☁️ 아이콘이 보이면 실행 중.
    Mac: 상단 메뉴바 오른쪽에 구름 ☁️ 아이콘이 보이면 실행 중.

    아이콘이 없다면: OneDrive 다운로드 페이지에서 앱을 설치하고 계정으로 로그인합니다.

  • OneDrive 폴더 안에 새 폴더 생성: Deeper Salon Leader

    Windows: 탐색기 → 왼쪽 목록의 OneDrive → 빈 곳 우클릭 → "새로 만들기" → "폴더" → 이름 입력.
    Mac: Finder → 왼쪽 사이드바 OneDrive → 빈 곳 우클릭 → "새로운 폴더" → 이름 입력.

    💡 멤버들은 "Deeper Salon" 폴더를 씁니다. 경선 씨는 "Deeper Salon Leader"로 분리해 리더 전용 데이터를 구분합니다.

STEP 2 — Obsidian Vault 생성

  • Obsidian 열기 → "Create new vault" 클릭

    이미 다른 Vault가 있다면, 왼쪽 아래 금고(🏛️) 아이콘을 클릭해 Vault 관리 화면으로 돌아갑니다.

  • Vault 정보 입력

    · Vault name: deeper-leader
    · Location: "Browse" 클릭 → 탐색기(또는 Finder)에서 방금 만든 OneDrive/Deeper Salon Leader 폴더 선택 → "Create" 클릭.

    ✅ 확인: OneDrive 폴더를 열어보세요. "Deeper Salon Leader" 안에 "deeper-leader" 폴더가 생겼으면 성공입니다.

STEP 3 — 야간비행님 공유 폴더 수락

야간비행님이 혼디 × 디퍼 협업 OneDrive 폴더를 공유했습니다. 이메일 또는 카카오톡으로 받은 공유 링크를 열어 수락하면 경선 씨 OneDrive에도 이 폴더가 동기화됩니다.
  • 야간비행님에게 받은 OneDrive 공유 링크를 브라우저에서 엽니다

    링크를 열면 OneDrive 웹 화면이 열립니다. 상단에 "내 OneDrive에 추가" 또는 "Add to my OneDrive" 버튼이 있습니다. 클릭합니다.

  • PC OneDrive 폴더에 "혼디 × 디퍼 협업" 폴더가 동기화됩니다

    탐색기(또는 Finder)에서 OneDrive 폴더 안에 "혼디 × 디퍼 협업" 폴더가 생겼는지 확인합니다. 처음에는 동기화에 1~2분이 걸릴 수 있습니다.

    ✅ 이 폴더가 야간비행님과의 협업 허브입니다. 두 사람 모두 이 폴더를 실시간으로 공유합니다.

리더 전용 Daily Note 템플릿 만들기

왜 템플릿이 필요한가: 매일 빈 화면을 마주하면 "뭐부터 쓰지"라는 마찰이 생깁니다. 이 마찰이 습관을 방해합니다. 템플릿이 있으면 노트를 열자마자 오늘 날짜와 섹션들이 준비돼 있고, 그냥 쓰기 시작하면 됩니다.
  • Templates 폴더 우클릭 → "New note" → 이름: Daily-Template
  • 오른쪽 편집 화면에 아래 내용을 그대로 복사해서 붙여넣기

    <% %>로 감싼 부분은 Templater 코드입니다. 노트를 열 때 오늘 날짜로 자동 교체됩니다. 수정하지 마세요.

    --- date: <% tp.date.now("YYYY-MM-DD") %> week: <% tp.date.now("W") %> type: daily role: leader --- # <% tp.date.now("MM월 DD일 (ddd)") %> ## 오늘의 질문 > ## 모닝페이지 메모 (오늘 쓴 것 중 한 줄만) ## 오늘의 키씬 — 가장 중요한 한 가지 - [ ] ## Claude와 탐구한 것 (질문을 가져가서 나눈 것 중 남길 문장) ## 오늘 발견한 것 ## 감사 + 비전 - 감사: - 비전: --- ## 리더 시선 (선택 — 세션 있는 날이나 협업 체크 날) (멤버들에게서 보이는 것, 혼디 협업 메모, 살롱에서 떠오른 것)
  • Daily Notes 설정에 템플릿 연결

    ⚙️ 설정 → Daily notes → "Template file location"에 Templates/Daily-Template 입력

템플릿이 적용되지 않을 때: Ctrl+P (Mac: Cmd+P) → "Templater: Open Insert Template modal" 입력 → "Daily-Template" 선택으로 수동 적용합니다.

Projects 설정

멤버들은 나 소개 카드를 매번 붙여넣지만, 경선 씨는 Projects를 써서 대화 맥락을 자동으로 유지합니다. 역할별로 프로젝트 3개를 만듭니다.

Projects 만드는 법: claude.ai 접속 → 왼쪽 사이드바 "Projects" → "+ New project" → 이름 입력 → 생성 후 "Project instructions" (또는 "Instructions") 영역에 아래 내용 붙여넣기
프로젝트 ① — 나의 탐구
일간 질문 탐구, 원천 텍스트 발굴 전용
나는 이경선이야. '디퍼 언브랜딩살롱 1기'를 운영하는 리더이자, 동시에 같은 여정을 걷는 실천자야. Trivium(문법→논리→수사) × AI를 통해 나만의 원천 텍스트를 발굴하고, 지속 가능한 콘텐츠 비즈니스 구조를 만드는 3개월이 목표야. 지금은 5월 — Grammar 단계. 매일 질문 하나를 깊이 탐구하는 중이야. 도움을 요청할 때: - 정답보다 더 깊은 질문으로 이끌어줘 - 내 언어 패턴이나 반복 주제가 보이면 짚어줘 - 리더로서의 관찰이 내 탐구에 영향을 줄 수 있어. 그것도 재료로 삼아줘
프로젝트 ② — 살롱 운영
세션 준비, 커리큘럼, 멤버 관찰, 인사이트 공유회 진행
나는 '디퍼 언브랜딩살롱 1기' 리더 이경선이야. 참여자 7명: 솔담, 햇살, 스윗워러, 야간비행, 헤더, 단무지, 퍼핀. 세션: 매주 목요일 밤 10시 줌. 세션 안에 인사이트 공유회 포함 — 각자 이번 주 발견한 것을 나누는 시간. 커리큘럼: 5월(문법·해체) → 6월(논리·구조) → 7월(수사·발화). 살롱 운영 도움을 요청할 때: - 세션 준비 질문 설계, 인사이트 공유회 진행 방식에 집중해줘 - 멤버 관찰이나 그룹 패턴 분석을 요청하면 솔직하게 말해줘 - 야간비행님은 혼디 프로젝트 대표이사이기도 해. 이 맥락을 기억해줘
프로젝트 ③ — 혼디 × 이경선 협업
야간비행/혼디 프로젝트 공동 작업 전용
이경선(디퍼살롱 리더)과 야간비행(혼디 대표)의 협업 채널이야. 혼디: 제주어로 "함께". 제주 공동체 프로젝트. 협업 범위: 트리비움 기반 혼디 집필, 강의 기획, 기획 제안서, 디퍼 연계 프로그램. 이 프로젝트에서: - 이경선의 트리비움 방법론과 야간비행의 제주 현장 언어가 함께 살아있어야 해 - 트리비움 프레임(문법→논리→수사)으로 혼디 콘텐츠를 구조화하는 게 목표야 - 결과물이 혼디의 나다움을 잃지 않도록 잡아줘
