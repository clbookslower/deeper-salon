01
Concept

이 시스템이 하는 일

왜 이 구조인가

살롱 기록은 지금 세 곳에 흩어져 있어요. 긱어스 게시판, 목요 공유회 자료, 카카오톡 대화. 각각은 살아있지만 연결되지 않아요. 시간이 지나면 사라지거나, 찾기 어려워지거나, 패턴이 보이지 않게 돼요.

이 시스템은 세 가지를 해요. 수집 — 흩어진 기록을 한 폴더로 모읍니다. 보존 — 마크다운 파일로 저장되어 어떤 기기, 어떤 앱으로도 영구적으로 열려요. 해석 — Claude가 쌓인 텍스트를 읽고 패턴과 인사이트를 추출해요.

📂
Obsidian
모든 텍스트가 머무는 집. 로컬 마크다운 파일이라 어떤 기기에서도 열리고, 영구적이에요.
Storage · Archive
☁️
OneDrive
팀 볼트를 모든 기기에서 실시간 동기화해요. 누가 어디서 써도 같은 파일을 봐요.
Sync · Share
⚡
Claude Desktop
볼트 전체를 읽고 패턴을 분석해요. 질문하면 축적된 기록을 바탕으로 답해요.
AI · Insights
핵심 원리
Obsidian은 지도를 그리고, Claude는 그 지도를 읽는다
노트끼리 [[링크]]로 연결되면 Obsidian Graph View에서 시각적 지도가 만들어져요. 누가 어떤 키워드를 자주 쓰는지, 어떤 개념이 반복되는지, 팀 안에서 어떤 생각들이 묶이는지가 보여요. Claude는 이 지도와 텍스트 전체를 읽어 "3주차에 팀 전체에서 가장 많이 등장한 키워드"처럼 사람이 직접 읽어서는 찾기 어려운 패턴을 뽑아줘요.

텍스트가 쌓일수록 지도가 정교해지고, 인사이트가 깊어져요. 12주가 끝날 때쯤이면 살롱 전체의 사유 지형도가 만들어지는 셈이에요.

볼트 구조 — 팀 + 개인

분리되면서도 교차되는 구조

03
Tools

세 가지 도구의 역할

각각 무엇을 하고 왜 필요한가

  • ①
    Obsidian — 텍스트의 집
    모든 노트가 .md(마크다운) 파일로 저장돼요. 서버가 없어요. 내 OneDrive 폴더 안의 파일이 전부예요. 어떤 기기, 어떤 앱으로도 열려요. 앱이 없어지거나 서비스가 종료돼도 파일은 남아요. 노트 안에 [[링크]]를 걸면 노트끼리 연결되고, Graph View에서 지도가 만들어져요. 팀원은 이 앱만 설치하면 돼요. 무료예요.
  • ②
    OneDrive — 동기화 통로
    팀 볼트를 공유 폴더에 위치시키면 팀원 전체가 실시간으로 같은 파일을 봐요. Mac, Windows, iPad 모두 OneDrive 앱만 설치하면 자동 동기화돼요. 개인 볼트는 각자의 개인 OneDrive에 위치 — 본인만 접근 가능해요. Windows에 기본 포함. Mac/iPad는 앱 설치. 기본 5GB 무료.
  • ③
    Claude Desktop — 두뇌
    MCP(Model Context Protocol) 연결로 Obsidian 볼트를 직접 읽고 써요. "이번 주 팀 포스팅 요약해줘", "내 모닝페이지 3주 패턴 알려줘" 같은 자연어 질문에 답해요. 볼트 전체가 컨텍스트이기 때문에 일반 Claude 채팅보다 훨씬 구체적이고 정확한 답이 나와요. 운영자(경선님)만 Claude Pro/Max 구독 필요. 팀원은 불필요.
  • +
    Obsidian 플러그인 3개 (무료)
    Local REST API — Claude Desktop이 볼트를 읽는 통로.
    Templater — 노트 템플릿 자동 적용. 데일리 노트를 열면 정해진 형식이 자동으로 채워져요.
    Dataview — 노트를 데이터베이스처럼 쿼리해요. "이번 달 아티스트 데이트 횟수" 같은 통계를 볼트 안에서 바로 볼 수 있어요.
04
Devices

기기별 역할

Mac · Windows · iPad — 뭘 써도 같은 볼트

볼트가 OneDrive에 있어서 기기는 그냥 접속 단말이에요. 팀원이 Mac을 쓰든 Windows를 쓰든 iPad를 쓰든, Obsidian 앱 하나만 설치하면 같은 파일을 봐요. 기기에 따라 할 수 있는 것이 조금 달라질 뿐이에요.

기기 주요 역할 가능한 것
Mac 추천 메인 작업 + 자동화 Obsidian 전체 기능 · Claude Desktop MCP · Claude Code 터미널 자동화
Windows 메인 작업 Obsidian 전체 기능 · Claude Desktop MCP · 자동화는 Mac보다 제한적
iPad 입력 · 열람 Obsidian 모바일로 노트 작성 · 읽기 · OneDrive 동기화. 무거운 분석은 PC에서.
Mac Mini 선택 자동화 서버 항상 켜둔 상태로 주간 리포트 자동 생성 · 정기 인사이트 추출 · Claude Code 스케줄 실행
팀원에게 필요한 것: Obsidian 설치 (무료) + OneDrive 동기화 설정 (무료) — 이게 전부예요. Claude나 유료 플랜은 팀원에게 불필요해요.
05
Setup

설치 순서

운영자 기준 — Windows / Mac 공통

  • 1
    Obsidian 설치 + 볼트 두 개 만들기
    obsidian.md 에서 앱 다운로드 (무료).
    OneDrive 안에 디퍼-팀 폴더 생성 → "Open folder as vault"로 열기.
    개인 OneDrive에 내-디퍼 폴더 생성 → 같은 방법으로 개인 볼트 열기.
    팀 볼트 폴더 안에 /daily /thursday /weekly /insights /books 폴더 생성.
  • 2
    팀 볼트 OneDrive 공유 링크 생성
    OneDrive에서 디퍼-팀 폴더 우클릭 → "공유" → "편집 가능한 링크 복사".
    팀원들에게 이 링크 공유 → 각자 "내 파일에 추가" 클릭하면 동기화 시작.
    이후 팀원이 Obsidian에서 이 폴더를 vault로 열면 완료.
  • 3
    플러그인 설치 (팀 볼트 기준)
    Obsidian 설정 → 커뮤니티 플러그인 → 검색 후 설치 · 활성화:
    Local REST API — Claude 연결 통로. 설치 후 API 키 메모해두기.
    Templater — 데일리 노트 자동 형식 적용.
    Dataview — 볼트 내 통계 쿼리.
  • 4
    Claude Desktop 설치 + MCP 연결
    claude.ai/download에서 앱 다운로드 (Pro 또는 Max 플랜 필요).
    Claude Desktop 설정 → claude_desktop_config.json 파일 열기.
    아래 내용 추가:
    {"mcpServers": {"obsidian": {"command": "npx", "args": ["-y", "obsidian-mcp"], "env": {"OBSIDIAN_API_KEY": "여기에 API 키", "VAULT_PATH": "볼트 경로"}}}}
    Claude Desktop 재시작 → 채팅창에 볼트 이름이 보이면 연결 완료.
  • 5
    CLAUDE.md 작성 — 가장 중요한 파일
    팀 볼트 루트에 CLAUDE.md 파일 생성.
    Claude Desktop이 매 세션 자동으로 이 파일을 읽어요. 여기에 살롱의 목적, 폴더 구조, 멤버 이름, 주간 리포트 형식을 적어두면 매번 설명 안 해도 돼요.
    이 파일이 시스템의 두뇌예요. 처음 한 번 잘 써두면 이후 모든 Claude 질문의 질이 달라져요.
막히는 지점: 4번 JSON 설정이 처음엔 낯설 수 있어요. 화면을 공유하거나 캡처를 보내주시면 바로 해결해드릴 수 있어요. 전체 설치 시간 약 1–2시간.

07
Insights

인사이트 추출 방법

Claude에게 어떻게 물어보는가

Claude Desktop에서 자연어로 질문하면 돼요. 볼트 전체가 컨텍스트로 들어가 있어서 축적된 기록을 바탕으로 구체적으로 답해요. 아래는 실제로 쓸 수 있는 질문 예시예요.

팀 분석
"3주차 팀 포스팅에서 가장 많이 등장한 키워드와 감정 톤 분석해줘"
팀 볼트 /daily 폴더의 해당 주 파일을 전체 읽고 단어 빈도와 정서 패턴을 정리해요. Graph View 링크 추천도 함께 줘요.
개인 패턴
"내 모닝페이지 최근 4주를 읽고 반복되는 주제와 변화 흐름 알려줘"
개인 볼트 /morning 폴더 전체를 시간 순으로 읽고 주제 클러스터와 어조 변화를 분석해요. 본인도 몰랐던 패턴이 나와요.
교차 분석
"팀 공유 기록과 내 개인 기록에서 겹치는 고민이 있어? 팀과 나눌 만한 것 추려줘"
두 볼트를 동시에 읽고 개인 기록 중 팀의 흐름과 공명하는 부분을 찾아줘요. /publish 후보를 제안해요.
주간 리포트
"이번 주 팀 볼트 전체 기반으로 주간 리포트 작성해서 /weekly에 저장해줘"
팀 포스팅 + 공유회 내용을 요약하고, 반복 테마, 눈에 띄는 성장, 다음 주 제안을 포함한 리포트를 직접 파일로 저장해요.
12주 결산
"살롱 12주 전체 아카이브를 읽고 팀의 사유 지형도를 정리해줘. 처음과 지금의 변화도 포함해서"
볼트 전체를 시간 순으로 읽고 팀 전체의 개념 지도, 개인별 변화 궤적, 공통으로 탐구한 주제를 종합 리포트로 만들어요.
Graph View 활용
노트가 쌓일수록 지도가 만들어진다
Obsidian Graph View에서 볼트의 모든 노트 연결을 시각화해요. 같은 키워드를 다른 날 다른 사람이 쓰면 노드가 연결돼요. 팀 안에서 어떤 개념이 중심에 있고, 누가 어디서 비슷한 생각을 했는지가 한눈에 보여요.

12주 후엔 이 지도가 살롱의 집단 지성 지형도가 돼요. 클로징 세션에서 이 지도를 함께 보는 것 자체가 하나의 리추얼이 될 수 있어요.
08
Cost

비용 구조

운영자만 유료, 팀원은 무료

도구 운영자 (경선님) 팀원 비고
Obsidian 무료 무료 개인 사용 무료. 팀 동기화는 OneDrive로 대체.
OneDrive 무료 (5GB) 무료 볼트 용량 초과 시 Microsoft 365 구독 고려.
Obsidian 플러그인 무료 무료 Local REST API · Templater · Dataview 모두 무료.
