01

Obsidian이 뭐야?

한 줄로 말하면: 내 컴퓨터에 저장되는 노트 앱이에요

보통 노트 앱Obsidian
저장 위치앱 서버 (남의 컴퓨터)내 기기 / 내 OneDrive
파일 형식앱 전용 형식.md 텍스트 파일
앱 없어지면기록 사라질 수 있음파일은 영원히 남음
노트 간 연결없음[[링크]]로 연결 가능
비용대부분 무료무료
왜 .md 파일이 좋아?
내 글이 진짜로 내 것이 되는 형식
.md는 마크다운(Markdown) 파일이에요. 그냥 텍스트 파일이에요. 메모장으로도 열려요. Obsidian이 없어져도, 10년 후에도, 어떤 앱으로도 열 수 있어요. 서버에 종속되지 않고, 내 OneDrive 폴더 안의 파일이 전부예요.

볼트 만들기

볼트(Vault) = 내 노트들이 모여있는 폴더

OneDrive에 만들어두면 iPad ↔ PC가 자동으로 동기화돼요.

iPad에서
  1. 1파일 앱 → OneDrive 폴더
  2. 2내-기록 폴더 새로 만들기
  3. 3Obsidian → Open folder as vault
  4. 4OneDrive → 내-기록 폴더 선택
PC에서
  1. 1OneDrive 폴더 안에 내-기록 폴더 생성
  2. 2Obsidian 열기 → Open folder as vault
  3. 3내-기록 폴더 선택 → 완료
이 폴더 안에 만드는 모든 노트가 iPad와 PC에서 동시에 보여요.

노트끼리 연결하기

이게 Obsidian의 진짜 힘이에요

[[ 입력하면 노트 목록이 뜨고, 고르면 연결돼요.

오늘 모닝페이지에서 [[언브랜딩]] 개념이 계속 떠올랐다. 지난주에 읽은 [[레베카 솔닛 메모]]랑 연결되는 느낌.
Graph View — 생각의 지도
링크가 쌓이면 내 사유 지형도가 만들어져요
왼쪽 사이드바 그래프 아이콘 클릭 → Graph View 열기.
같은 키워드를 다른 날 연결하면 노드가 생겨요. 처음엔 많이 연결 안 해도 돼요. 생각날 때만 해도 돼요. 조금씩 쌓이다 보면 어느 날 지도가 보여요.

10

자주 묻는 것들

Q실수로 노트 삭제했어요.
Obsidian 왼쪽 사이드바 → 휴지통 아이콘에서 복구할 수 있어요. OneDrive에서도 복구 가능해요.
Q다른 사람이 내 노트를 볼 수 있어요?
개인 OneDrive에 저장하면 본인만 봐요. 공유 폴더에 올리지 않는 한 아무도 못 봐요.
Q노트가 너무 많아지면 어떻게 관리해요?
검색(Ctrl+O 또는 모바일 🔍)이 가장 빠르고 편해요. 폴더 구조 완벽하게 안 잡아도 검색으로 다 찾아요.
Q마크다운 문법 다 외워야 해요?
아니요. # 제목이랑 - 목록, 이 두 개만 알아도 충분해요. 나머지는 필요할 때 찾아쓰면 돼요.
Q폰트나 테마 바꾸고 싶어요.
설정(⚙️) → 외관(Appearance) → 테마(Themes)에서 다양한 테마 설치 가능해요. 무료예요.
